# GrowthFit - Kids Fitness Screening System
**ระบบคัดกรองเตรียมความพร้อมในการออกกำลังกายเพื่อกระตุ้นความสูงสำหรับเด็ก**

แอปพลิเคชันรูปแบบ Web Application สำหรับคัดกรองและประเมินสมรรถภาพทางกายของเด็ก เพื่อเตรียมความพร้อมก่อนการออกกำลังกายกระตุ้นความสูง ตัวแอปพลิเคชันรองรับการใช้งานทั้งบนหน้าจอคอมพิวเตอร์ (PC) และโทรศัพท์มือถือ (Responsive Design)

## ✨ คุณสมบัติหลัก (Features)
- **ระบบคำนวณอายุอัตโนมัติ**: ระบบรับค่าอายุและจัดกลุ่มเด็กอัตโนมัติ (กลุ่ม 1: 4-7 ปี, กลุ่ม 2: 8-14 ปี) 
- **ระบบคำนวณ BMI**: ประมวลผลดัชนีมวลกายจากน้ำหนักและส่วนสูงโดยอัตโนมัติ
- **Dynamic Dashboard**: แสดงรายการทดสอบสมรรถภาพทางกายที่แตกต่างกันตามกลุ่มอายุ
- **Responsive Design**: ออกแบบ UI สไตล์ Premium Glassmorphism ที่สวยงามและรองรับการแสดงผลทุกขนาดหน้าจอ
- **ระบบออกรายงาน (PDF Report)**: สร้างสรุปผลการประเมินในรูปแบบ PDF ที่อ่านง่าย พร้อมนำไปใช้งานต่อ

## 👥 กลุ่มเป้าหมาย (Target Groups)
ระบบจะแบ่งผู้ใช้ออกเป็น 2 กลุ่มหลัก:

### 👶 กลุ่มที่ 1: อายุ 4 - 7 ปี (Play-based Developmental Screen)
เน้นความปลอดภัย ความสนุก และทักษะการเคลื่อนไหวพื้นฐาน (fundamental movement skills) ใช้เวลาสั้น หลีกเลี่ยงแรงกระแทกสูง วัด 4 รายการ: 
1. น้ำหนักตามเกณฑ์ส่วนสูง
2. ยืนทรงตัวขาเดียว (Single Leg Stance)
3. กระโดดขาเดียว (Single-leg Hop)
4. กระโดดไกลอยู่กับที่ (Standing Long Jump)

### 🧒 กลุ่มที่ 2: อายุ 8 - 14 ปี
เน้นบททดสอบที่หลากหลายและใช้ความแข็งแรงทนทานมากขึ้น จำนวน 7 รายการ:
1. BMI (ดัชนีมวลกาย)
2. 10x5 m Shuttle (วิ่งกลับตัว 10x5 เมตร)
3. 3 Min Step Test (ทดสอบก้าวขึ้น-ลง 3 นาที)
4. Standing Long Jump (กระโดดไกลอยู่กับที่)
5. Handgrip (วัดแรงบีบมือ)
6. Sit & Reach (นั่งงอตัวไปข้างหน้า)
7. Single Leg Stance (ยืนทรงตัวขาเดียว)

## 📊 คู่มือเกณฑ์มาตรฐานสมรรถภาพทางกาย
รวบรวมค่าเฉลี่ยทางคลินิกวิชาชีพกายภาพบำบัดและเวชศาสตร์การกีฬา (อายุ 4 - 15 ปี)

### 1. การทดสอบยืนทรงตัวขาเดียว (Single Leg Stance Test: SLS)
- **อุปกรณ์:** พื้นเรียบ, stopwatch
- **ท่าทดสอบ:** single-leg stance 
- **Trials:** 2 ครั้งต่อข้าง
- **Score:** เวลา balance
- **การแปลผล:** นับเวลาวินาที เทียบกับค่ามาตรฐาน
- **Error:** ไม่บันทึก side-to-side asymmetry

### 6. ทดสอบก้าวขึ้น-ลง 3 นาที (3 Min Step Test)
ก้าวขึ้น-ลงแท่นตามจังหวะ 3 นาที ประเมินสมรรถภาพของระบบหัวใจและหลอดเลือด (Cardiorespiratory Fitness)
- **ดีมาก:** HR recovery ลดลง ≥30 bpm จาก HR peak
- **ดี:** HR recovery ลดลง 20–29 bpm
- **ปานกลาง:** HR recovery ลดลง 10–19 bpm
- **ต่ำ:** HR recovery ลดลง <10 bpm

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- **HTML5** สำหรับโครงสร้างแอปพลิเคชัน
- **Vanilla CSS3** สำหรับการจัดรูปแบบและการแสดงผล (ไม่ต้องพึ่งพา Framework ภายนอก)
- **Vanilla JavaScript** สำหรับการจัดการตรรกะคำนวณ (Logic) และการประมวลผลผลลัพธ์
- **html2pdf.js** สำหรับการ Export รายงานสรุปผลออกเป็นไฟล์ PDF

## 🚀 วิธีการใช้งาน (How to run)
โปรเจกต์นี้ทำงานฝั่ง Client-side 100% ไม่จำเป็นต้องติดตั้งเซิร์ฟเวอร์
1. เข้าไปที่โฟลเดอร์โปรเจกต์
2. ดับเบิลคลิกที่ไฟล์ `index.html` เพื่อเปิดผ่านเว็บเบราว์เซอร์ (แนะนำให้ใช้ Google Chrome, Safari หรือ Microsoft Edge)
3. เริ่มต้นใช้งานได้ทันที (สามารถย่อหน้าต่างเบราว์เซอร์เพื่อทดสอบมุมมองแบบหน้าจอมือถือได้)
